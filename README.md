# Bakka — AI-Powered Umrah & Hajj Platform

Bakka is a platform for Russian-speaking and CIS-region pilgrims planning Umrah and Hajj.

## What it does
- Aggregates verified tour operators and packages
- Provides AI-guided pilgrimage preparation in Russian
- Connects pilgrims with trusted agencies across CIS

## Stack
- React Native / Expo (mobile app)
- Node.js backend
- Telegram Mini App

## Status
🚧 In active development

---
*Bakka — from the Quranic name for Mecca (Surah 3:96)*
